js 模块化的学习

es6与commonJs模块加载的区别

es6为ECMAScript2016标准,commonJs为此前为解决模块化的模块加载器.

es6的模块输出为引用 , commonJs的模块输出为值.

循环加载下,commonJs是加载时执行,而es6只是生成被加载模块的引用.

amd 与 cmd

amd:异步模块规范, cmd:通用模块规范

amd 是requireJs 产出的,cmd 的是seaJs产出的

amd 提倡预先执行, cmd 提倡懒执行

seaJs注重代码调试, 依赖spm打包

模块加载器出现前的问题

  1. 一次加载多个js文件, 加载的时候, 浏览器会停止对网页的渲染.

  2. js文件间存在依赖关系, 必须把依赖最大的放在最后.
