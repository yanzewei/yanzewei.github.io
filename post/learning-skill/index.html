learning skill

Oct 31, 2016

自律

  1. 心理暗示.写在手上,或床头,或桌上.

  2. 规律作息,最迟9点要出门吃早餐.

  3. 合理运动,每周至少两次去健身房,但不能影响到学习睡眠.

  4. 屏蔽干扰,关掉手机wifi.

学理论时

  1. 带着解决问题的心思去收集材料, 不限一本书, 不限一个作者.

  2. 读资料, 或解题时, 有时可以想象出一个图, 研究它

  3. 相关的概念, 术语与释义熟记

  4. 假设, 变量与条件

心态

学习效果不能立刻让大脑感受到奖励, 但会有那么一刻, 如果想放弃, 就回想过去的某些时刻. 获得了成功后, 我简直觉得自己就是天才, 今后什么困难我都没问题, 继续下去心情也是愉悦的.

同时也起回想下, 在过去某一时刻坚持了那么久, 当中出现了失败, 靠的是什么在支撑. 因为在我的努力下, 我在窃喜某个小方面, 我又掌握熟悉了, 甚至绝对做得到满分.
