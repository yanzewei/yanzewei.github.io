在n个物品中挑选若干物品装入背包，最多能装多满？假设背包的大小为m，每个物品的大小为A[i]

###样例 >如果有4个物品[2, 3, 5, 7]

如果背包的大小为11，可以选择[2, 3, 5]装入背包，最多可以装满10的空间。

如果背包的大小为12，可以选择[2, 3, 7]装入背包，最多可以装满12的空间。

函数需要返回最多能装满的空间大小。

###注意 >你不可以将物品进行切割。

c语言实现代码：

###思路 >取n个物品的最优值S[n]，那先想想取前n-1件物品的最优值，设为S[n-1]，如果S[n-1]+A[n]>m，那么S[n]=S[n-1]; >取0个物品的最优值S[0],应该为0，以它为初始值； >递归的去遍历这个过程，取j=0,j++到m,记S[i][j],表示取前i件物品填充j大小的空间的最优值。

###C语言实现代码 >#include >#include >int main() >{ > int i,j,n,m,*A,**S; > printf(“input backage weight:\n”); > scanf(“%d”,&m); > printf(“input items number:\n”); > scanf(“%d”,&n); > printf(“input each item weight:\n”); > A = (int *)malloc(n*sizeof(int)); > S = (int **)malloc((n+1)*sizeof(int * )); > S[0] = (int *)malloc(m*sizeof(int)); > for(i=1;i<=n;i++) > {
> scanf(“%d”,&A[i-1]); > S[i] = (int *)malloc(m*sizeof(int)); > }
> for(j=0;j<=m;j++) > {
> S[0][j] = 0; > }
> for(i=1;i<=n;i++) > {
> for(j=0;j<=m;j++) > {
> S[i][j] = S[i-1][j]; > if(j>=A[i-1]) > {
> if(S[i][j-A[i-1]] + A[i-1] > S[i][j]) > {
> S[i][j] = S[i][j-A[i-1]] + A[i-1]; > }
> } > } > } > printf(“max weight:%d\n”,S[n][m]); > return 0; >} >
